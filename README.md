# Projeto1-teste-login-auto

Testes E2E automatizados de login no site automationpractice.com.br usando **CodeceptJS + Playwright**. Projeto para estudos em QA/Automação.

[![Status](https://github.com/ThomasSaraiva-ops/Projeto1-teste-login-auto/actions/workflows/ci.yml/badge.svg)](https://github.com/ThomasSaraiva-ops/Projeto1-teste-login-auto/actions)

## 🚀 Funcionalidades Testadas
- Login bem-sucedido
- Validações de erro: email vazio, senha vazia, só email/senha

## 📋 Pré-requisitos
- Node.js 18+
- Git

## 🛠️ Instalação
1. Clone: `git clone https://github.com/ThomasSaraiva-ops/Projeto1-teste-login-auto.git`
2. Entre: `cd Projeto1-teste-login-auto`
3. Instale: `npm install`
4. Browsers: `npx playwright install`

## ▶️ Executar Testes
- npx codeceptjs run
